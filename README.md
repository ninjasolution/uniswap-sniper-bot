## Instalación/Installation

- Requiere/requires [Node.js](https://nodejs.org/) v12+

```sh
git clone https://github.com/ninjasolution/uniswap-sniper-bot.git
cd uniswap-sniper-bot
npm install
npm start
```

Para abrir con el devtool...

```sh
npm star dev
```
